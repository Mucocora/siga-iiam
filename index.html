Backup dos dados

Os dados (planos, produção e listas padrão) ficam guardados no navegador (localStorage) deste computador. Exporte um ficheiro de backup regularmente ou antes de mudar de computador/navegador. Cada colega que abrir este ficheiro tem os seus próprios dados isolados — não há sincronização automática entre pessoas nesta versão de protótipo. O backup inclui os utilizadores registados (incluindo a palavra-passe simulada em texto simples) — guarde o ficheiro com cuidado.
Protótipo interno IIAM — dados guardados apenas neste navegador. Não substitui o sistema oficial.
